# Subir o editar contenido en la web
#
En busca de una mejor solución para manejar el contenigo de la página web de OPADES, fue tomada la decisión de  migrar el contenido de la misma a una nueva plataforma:  [Rowy]("https://rowy.app/auth?redirect=%2F").
Rowy nos permite manejar el contenido de la página web de una manera más sencilla y eficiente.  Para poder subir o editar contenido en la página web, es necesario tener una cuenta en Rowy.

Para obtener una cuenta favor hablar con el encargardo de la web.

En el siguiente tutorial podemos ver como hacer cambios en la página web.

  Aclaración: Los cambios efectuados se reflejan en alrededor de 1 minuto, no media hora.

  Subir formularios: Para subir formularios pueden seguir el ejemplo de los existentes, de esa manera cualquier persona va a poder acceder a los mismos con el SLUG correspondiente. Pero para que el formulario aparezca en el menú de navegación de la página tiene que incluirse manualmente, por esa razón si desean incluir un formulario nuevo favor comuniquense con el administrador de la web.
