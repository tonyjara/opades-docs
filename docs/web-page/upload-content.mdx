# Subir contenido a la Web

La mayoria del contenido de la página de Opades es alimentado a través de una página en Wordpress. Es decir, para poder editar el contenido en Opades tenemos que acceder primero a la página en  [WordPress]("https://wp.opades.org.py/wp-login.php").

Para solicitar una cuenta favor hablar con el encargardo de la web.
