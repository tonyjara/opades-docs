# Subir o editar contenido en la web
#
En busca de una mejor solución para manejar el contenigo de la página web de OPADES, decidi migrar el contenido de la misma a una nueva plataforma:  [Rowy]("https://rowy.app/auth?redirect=%2F").
Rowy nos permite manejar el contenido de la página web de una manera más sencilla y eficiente.  Para poder subir o editar contenido en la página web, es necesario tener una cuenta en Rowy.

Para obtener una cuenta favor hablar con el encargardo de la web.
